WebRTC Voice Client

Capture microphone, stream to the Voila backend (Sarvam STT + Groq LLM + Sarvam TTS), play response. Start Voila: go run ./cmd/voila -config config.json with transport: "both". Serve this page: cd tests/frontend && python -m http.server 3000 then open http://localhost:3000/webrtc-voice.html. Set Backend URL to where Voila runs (e.g. http://localhost:8080).

idle

Pipeline: mic → STT → LLM → TTS → speaker. Mic sent as audio track; TTS received on remote track.
