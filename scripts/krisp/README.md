# Krisp Viva

Krisp Viva integration and tests are maintained in the upstream Python repo (scripts/krisp).

This directory is a placeholder for future Krisp support in this (Go) codebase. The upstream scripts include:

- `audio_file_utils.py` – audio file handling for tests
- `test_krisp_viva_filter_audiofile.py`
- `test_krisp_viva_turn_audiofile.py`

For running Krisp Viva tests, use the upstream project.
