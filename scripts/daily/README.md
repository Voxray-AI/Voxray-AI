# Daily / Tavus transport

This repository does **not** implement the Daily or Tavus transport. Transport tests and scripts for Daily/Tavus live in the upstream Python Pipecat repo:

- [pipecat-ai/pipecat – scripts/daily](https://github.com/pipecat-ai/pipecat/tree/main/scripts/daily)

Notable script: `test_tavus_transport.py` (Python, requires Daily/Tavus and the Pipecat Python SDK).

To run Daily or Tavus-based tests, use the upstream [pipecat-ai/pipecat](https://github.com/pipecat-ai/pipecat) project.
